Custom Subscriber Dashboard: Displays real-time data from 5 Arduino nodes located at different sites. Each node publishes temperature to Adafruit IO; this dashboard subscribes to all 5 feeds simultaneously. Data updates appear in real-time as nodes send readings (staggered timing simulates independent Arduino devices).
